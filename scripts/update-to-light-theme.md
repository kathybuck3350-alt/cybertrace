# Theme Update Guide
Common replacements needed:
- bg-dark-900 → bg-white or bg-gray-50
- bg-dark-800 → bg-gray-50 or bg-white
- text-gray-100 → text-gray-900
- text-gray-300 → text-gray-700
- text-gray-400 → text-gray-600
- border-primary-500/20 → border-gray-200
- border-primary-500/30 → border-gray-300
- glass-effect dark versions → light versions
